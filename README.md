# ADA-Project-

This repository contains the final submission for the ADA Capstone Project.

**Repository Structure**

The repository is organised as follows:

- `Final Capstone Project ADA.ipynb`  
  Main Jupyter notebook containing all data processing, analysis, and results.

- `sp500.xlsx`, `MSCI.xlsx`, `ishares.xlsx`, `vix.xlsx`, `stoxx.xlsx`, `HFRXGL Index.xlsx`  
  Input datasets used in the analysis.

- `README.md`  
  Project documentation and execution instructions.

⚠️ The Excel files must remain in the repository root and must not be moved.

## How to Run the Project :

This repository contains the final notebook for the ADA Capstone Project
The repository must be kept unchanged after cloning or downloading:

**Instructions :** 

Clone the repository or download it as a ZIP file.
Open the repository folder.
Open Final Capstone Project ADA.ipynb from the repository root.
Run all cells from top to bottom.
No additional configuration or path modification is required.

The notebook uses relative paths and assumes execution from the repository root directory.
As long as the original repository structure is preserved, the code will run correctly on any system.

**Dependencies :** 

The project requires the following Python libraries:

- Python 3.9+
- pandas
- numpy
- matplotlib
- seaborn
- statsmodels
- scikit-learn
- openpyxl

**Reproducibility : **

All datasets are included in the repository, and all file paths are defined relative to the repository root.
As long as the repository structure is preserved, the results are fully reproducible on any system.
